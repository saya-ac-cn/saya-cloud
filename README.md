# saya-cloud

> consul 分支

以consul作为注册和配置中心的微服务架构

### 安装micro框架
```shell
# V4 版本
go get go-micro.dev/v4

# V3版本
go get github.com/asim/go-micro/v3
```

### 安装consul注册配置中心组件
```shell
# V4 版本
go get github.com/asim/go-micro/plugins/registry/consul/v4

# V3版本
go get  github.com/asim/go-micro/plugins/registry/consul/v3
```
